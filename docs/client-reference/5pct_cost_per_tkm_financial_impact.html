This is a preserved reference artifact for product implementation mapping. It intentionally keeps only the core formula summary to avoid duplicating an external source file in full.

Core formulas copied from client model

  • Total annual haulage cost = fleet size * all-in cost per truck
  • Direct annual saving = total haulage cost * 5%
  • Revenue equivalent = direct saving / EBITDA margin
  • Enterprise value created = direct saving * 6x to 8x EBITDA multiple

Default scenario in client file

  • Fleet size: 30 trucks
  • All-in truck cost: R44M per truck per year
  • EBITDA margin: 30%
  • Expected direct saving at 5%: R66M per year
